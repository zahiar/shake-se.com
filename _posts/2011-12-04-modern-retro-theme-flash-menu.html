Modern Retro is a unique pass protected menu with a simple design.

Unlike the other pass protected menus which require you to type in a numerical pass code, this menu uses the arrow keys instead. This menu also allows full protection, as the menu will only display once you have typed in the code.

The default pass code is in the readme.txt file inside the zip file, also if you would like to customise the passcode, simply leave a message, and I'll get one created for you. An example code would be, Up, Right, Left, Down.

As ever, this menu will work on both Flash Lite 2/3/Aino phones.
